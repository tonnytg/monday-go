# monday-go
This is a CLI for Monday write on GoLan!

## How to use?
1. Access your account on Monday

2. Get API Token

3. Export env `KEYMONDAY`
```export KEYMONDAY="XXXXXXXXXX"```

4. Execute monday-go --help
